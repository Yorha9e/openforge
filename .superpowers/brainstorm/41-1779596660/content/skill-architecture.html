▼ 启动扫描 + mtime watch 热重载 ▼
🔍 SkillLoader
├─ Scan()          扫描三层目录，解析 YAML frontmatter
├─ LoadIndex()     读 skill_config.yaml，合并运行时状态
├─ Watch()         mtime 轮询 5s，文件变更自动重载
├─ Match(req)      stage过滤 → 关键词打分 → 优先排序 → top-K
└─ sync.RWMutex   读多写少，Match 持读锁，Reload 持写锁

关键设计决策

并发RWMutex（进程内）+ atomic rename（文件写入）
热重载5s mtime 轮询，文件变更 → 写锁重载 → 不影响正在执行的 Match
优先级引擎独立 goroutine，每日 03:00 批量更新 skill_config.yaml
强制拉起user_command → 跳过匹配 → top-1 → 不占 token budget → WARN 日志
降级SkillLoader 故障 → 仅 Tool 注入，Skill 空列表，WARN 日志，不阻断 Pipeline
